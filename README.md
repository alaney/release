# UI Web Server Release

Assumes the following directory structure

```
/code
  /release
  /ui-webserver
  /odata
  /urrcs
  /deployment
```

Build dockerfile with
`docker build . -t release`

Run
`docker run --rm -it -v /mnt/c/Users/alaney/code:/code release`

Run the script from the `code` directory.
